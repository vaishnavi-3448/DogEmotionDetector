# 🐶 Paww AI – Dog Emotion Detection & Translator

## 📌 Overview

PawMood AI is a deep learning-based web application that analyzes a dog's facial expression and predicts its emotional state. The application also provides a fun "dog translation" feature that generates humorous thoughts based on the detected emotion.

The project uses Transfer Learning with MobileNetV2.

---

## ✨ Features

* Upload a dog image
* Detect dog emotions using Deep Learning
* Emotion categories:

  * Happy 😊
  * Sad 😢
  * Angry 😠
  * Relaxed 😌
* Confidence score for predictions
* Fun dog thought translator
* Paw Score generator
* Treat Probability estimator
* User-friendly web interface

---

## 🧠 Machine Learning Model

### Model Architecture

* MobileNetV2 (Transfer Learning)
* GlobalAveragePooling2D
* Dense Layer (ReLU)
* Softmax Output Layer

### Why MobileNetV2?

* Lightweight architecture
* Faster training
* Suitable for smaller datasets
* High accuracy through transfer learning

Dataset Source:

Roboflow Dog Emotion Classification Dataset

Classes:
